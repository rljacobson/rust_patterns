§Techniques for type-erasure.

The type-erased API module is an illustration of a type, a database index, having both a typed and type-erased API.

Sometimes you don’t have complete control over the type you want to expose. Suppose you want a type-erased interface to a type but not instances of that type. The static_interface module illustrates how to do this in a couple of different ways.
